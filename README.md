# AutoCare Web 🔧

Versão web do AutoCare — aplicativo de controle de manutenção veicular.

## 🚀 Tecnologias
- React
- TypeScript
- Vite
- Jest
- React Testing Library

## 🧪 Testes
O projeto é desenvolvido com TDD — testes escritos antes da implementação.

```bash
npm test
```

## ▶️ Como rodar localmente
```bash
npm install
npm run dev
```
