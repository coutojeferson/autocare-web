# AutoCare Web 🔧

Versão web do AutoCare — aplicativo de controle de manutenção veicular.

## 🚀 Tecnologias
- React
- TypeScript
- Vite
- Jest
- React Testing Library

## 🧩 Componentes
- `VehicleCard` — exibe nome e KM do veículo formatado
- `AddVehicleButton` — botão de adicionar veículo
- `VehicleTypeSelector` — seletor de tipo (carro/moto)
- `VehicleList` — lista de veículos com mensagem de estado vazio
- `SearchInput` — input de busca controlado
- `Counter` — contador com regra de não ir abaixo de zero

## 🧪 Testes
O projeto é desenvolvido com TDD — testes escritos antes da implementação.

```bash
npm test
```

## ▶️ Como rodar localmente
```bash
npm install
npm run dev
```
